Blender Qt3D Exporter

INSTALL

1) Clone repository locally

2) Create a zip file for the folder containing the blender exporter

3) Open up Blender, File -> User Preferences -> Add-ons -> Install Add-on From
File

4) Select the zip file you just created

5) Activate the add on (search for qt)

USAGE

Once installed you should have a new entry in the File -> Export menu -> Qt3D (.qml)

Select the folder to which you would like your Qt3D application to be
generated (filename is ignored, only directory matters)
